# Create Lib

Create single ts lib, test with jest, and publish with npm.

## Usage

1. Create

```bash
mkdir <libName>
cd <libName>
npm create @2snail/lib
-- or --
yarn create @2snail/lib
```

2. Install

```bash
npm i
-- or --
yarn install
```

3. Test

```bash
npm test
npm run test:watch
-- or --
yarn test
yarn test:watch
```

4. Build

```bash
npm run build
-- or --
yarn build
```

5. Publish

```bash
npm run publish
-- or --
yarn publish
```
